# 🤖 AGENT KIT — Bộ Công Cụ Mở Rộng Năng Lực AI Agent Toàn Diện

**Sản phẩm của Lam Sơn Edutech (LSE)**
**Phân khúc:** Dành cho Lập trình viên, Đội phát triển phần mềm, Doanh nghiệp công nghệ
**Tương thích:** Claude Code, Gemini, các AI Agent framework

---

## 1. Giới thiệu sản phẩm

**Agent Kit** là bộ công cụ mở rộng (toolkit) được thiết kế để tăng cường mạnh mẽ năng lực của các AI Agent trong phát triển phần mềm. Thay vì mỗi lập trình viên hay đội nhóm phải tự xây dựng từ đầu, Agent Kit cung cấp sẵn 20 AI Agent chuyên biệt, 39 skill module và 11 workflow tự động — có thể tái sử dụng ngay cho bất kỳ dự án nào.

Agent Kit được xây dựng dựa trên nền tảng Antigravity Kit và tích hợp các skill chọn lọc từ Superpowers (MIT License), đảm bảo chất lượng và tính ổn định khi triển khai thực tế.

---

## 2. Đối tượng phục vụ

| Đối tượng | Lợi ích |
|---|---|
| **Lập trình viên cá nhân** | Tăng tốc độ phát triển, có sẵn best practices cho mọi tình huống |
| **Đội phát triển (5–50 người)** | Chuẩn hóa workflow, đảm bảo chất lượng code đồng đều toàn nhóm |
| **Startup công nghệ** | Nhanh chóng MVP, tiết kiệm thời gian setup AI tools |
| **Doanh nghiệp Edtech / Fintech** | Bộ agent phù hợp từng domain (Frontend, Backend, Security, Mobile...) |

---

## 3. Thành phần bộ Kit

### 🧑‍💻 20 AI Agent Chuyên Biệt
Mỗi agent là một "chuyên gia AI" với vai trò và kiến thức riêng:
- **Frontend agents**: React, UI/UX, Tailwind, Web Design
- **Backend agents**: API Design, Node.js, Python, Database
- **Mobile agents**: Phát triển ứng dụng di động
- **Security agents**: Vulnerability Scanner, Red Team Tactics
- **Architecture agents**: App Builder, System Design, Planning

### 📚 39 Skill Module
Kiến thức chuyên sâu được đóng gói theo domain, bao gồm:
- `react-best-practices` · `api-patterns` · `database-design`
- `testing-patterns` · `tdd-workflow` · `code-review-checklist`
- `vulnerability-scanner` · `architecture` · `brainstorming`
- `subagent-driven-development` · `using-git-worktrees`
- Và 28 module khác bao phủ toàn bộ vòng đời phát triển phần mềm

### ⚡ 11 Workflow (Slash Commands)
Các quy trình tự động kích hoạt bằng lệnh đơn giản:
- `/create` — Tạo component/module mới theo chuẩn
- `/debug` — Phân tích và gợi ý sửa lỗi
- `/plan` — Lập kế hoạch phát triển tính năng
- `/test` — Tạo test cases tự động
- Và 7 workflow khác cho các tình huống phổ biến

### 🔍 20 Validation Script
Script tự động kiểm tra: code quality, security, performance, naming convention — chạy trước khi commit để đảm bảo tiêu chuẩn.

## 4. Cách sử dụng

```bash
# Clone và copy vào dự án trong 3 bước
git clone https://github.com/luonghaianh1208/Agent-Kit.git
cp -r Agent-Kit/.agent /path/to/your-project/
cp Agent-Kit/GEMINI.md /path/to/your-project/
```

Sau đó, AI Agent trong dự án của bạn tự động có đầy đủ năng lực từ 20 agents + 39 skills.
